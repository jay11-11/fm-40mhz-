关键连接点 (Critical Connection Points):
=========================================

1. VCO模块的 J1(+9V) ← FM发射机的9V电源
2. VCO模块的 J1(GND) ← FM发射机的GND
3. VCO模块的 J3(Pin 1) ← FM发射机的振荡器OUT (C7点)
4. VCO模块的 J3(Pin 2) ← FM发射机的GND


电压测试点 (Voltage Test Points):
==================================

TP1 (电位器输出): 应该是 0-8.5V 可调
TP2 (滤波输出): 应该是 0-8.5V 低纹波
TP3 (BB910控制): 应该是 0-8.5V 干净信号


频率调节步骤 (Frequency Adjustment Procedure):
===============================================

1. 用频率计连接到FM天线
2. 缓慢旋转R2电位器
3. 观察频率变化范围(应该是±2-5MHz)
4. 找到中心频率点(R2中点)
5. 用示波器测量波形质量
6. 微调R3值改善频率线性度


故障排查 (Troubleshooting):
==========================

问题: 频率不变
  → 检查J3连接
  → 测量控制电压是否变化
  → 检查BB910是否焊接牢固

问题: 频率调范太小
  → 检查BB910型号
  → 尝试调整R1值(改变分压比)
  → 增加C7电容的容值

问题: 杂音大
  → 增加C1滤波电容(试试22uF)
  → 缩短控制电压线长
  → 检查GND连接

问题: 频率跳变
  → 检查可变电阻接触是否良好
  → 增加C1滤波电容
  → 用示波器检查控制电压
