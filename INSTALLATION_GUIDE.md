# FM VCO 模块安装指南 (Installation Guide)

## 快速开始 (Quick Start)

### 步骤1: 准备工作

```
□ 准备焊接工具 (Soldering iron, solder, flux)
□ 准备万用表 (Multimeter)
□ 准备频率计 (Frequency meter) - 可选
□ 准备所有元器件 (See BOM.csv)
□ 准备PCB板
```

### 步骤2: 元器件焊接顺序

#### 2.1 焊接被动元件 (Passive Components)

```
1. R1(100K 1/4W电阻)
   ├─ 位置: 电源输入附近
   ├─ 方向: 任意
   └─ 焊接: 两端都要焊牢

2. R3(10K 1/4W电阻)
   ├─ 位置: BB910旁边
   ├─ 方向: 任意
   └─ 焊接: 两端都要焊牢

3. C1(10uF 电解电容)
   ├─ 位置: R3旁边
   ├─ 方向: 长脚接+9V,短脚接GND
   ├─ 标记: " - " 标记必须对应
   └─ 焊接: 注意极性!
```

#### 2.2 焊接可变电阻 (Potentiometer)

```
4. R2(100K 滑片式电位器)
   ├─ 位置: 单独区域,便于后续调整
   ├─ 引脚: 通常是3脚(两端 + 中点)
   ├─ 接线方式:
   │  ├─ 一端接 +9V (经过R1)
   │  ├─ 中点接 R3
   │  └─ 另一端接 GND
   └─ 焊接: 所有脚都要焊牢
```

#### 2.3 焊接变容二极管 (Varactor Diode) - 最后

```
5. D1(BB910 变容二极管)
   ├─ 位置: 最靠近输出端
   ├─ 极性: 黑色条 = K脚(正极)
   ├─ 引脚定义:
   │  ├─ K脚(正,黑条) → 到FM发射机OUT(C7)
   │  └─ A脚(负) → GND
   ├─ 焊接前:
   │  ├─ 放电(触摸金属物体)
   │  └─ 不要用手直接接触管脚
   └─ 焊接: 温度<350°C, 时间<3秒
```

#### 2.4 焊接接头 (Connectors)

```
6. 电源接头 J1 (2-Pin)
   ├─ Pin 1: +9V 红线
   └─ Pin 2: GND 黑线

7. 输出接头 J2 (2-Pin)
   ├─ Pin 1: 控制电压 (从R2中点)
   └─ Pin 2: GND

8. 振荡器接头 J3 (2-Pin)
   ├─ Pin 1: 连接到FM发射机C7
   └─ Pin 2: GND
```

### 步骤3: 焊接完成后检查

```
□ 检查所有焊点是否饱满
□ 检查是否有冷焊点(光亮的、银灰色)
□ 检查是否有焊锡桥接(短路)
□ 用万用表检测 +9V 到 GND 的电阻(应该>1K欧)
```

## 连接到FM发射机 (Connection to FM Transmitter)

### 方式A: 外部模块 (Recommended for testing)

```
FM发射机板 ←→ 连接线 ←→ VCO模块

          焊点/连接

J1(+9V) ←─→ FM板+9V
J1(GND) ←─→ FM板GND
J3(Pin1)←─→ FM板C7点(振荡器OUT)
J3(Pin2)←─→ FM板GND
```

**具体步骤:**

1. 关闭FM发射机电源
2. 用导线连接VCO模块J1到FM发射机电源
3. 用导线连接VCO模块J3到FM发射机的C7点
4. 确保所有GND连接都接在一起
5. 打开电源,测试

### 方式B: 直接集成 (For permanent installation)

```
1. 找到FM发射机的C7电容(18-22pF)
2. 用BB910替代它,或在它旁边并联BB910
3. 从BB910接一根线到电位器
4. 用滤波网络连接到电位器
5. 用电源线连接9V和GND
```

## 测试程序 (Testing Procedure)

### 第1步: 电压测试

```bash
工具: 万用表

1. 打开电源
   万用表黑表笔接GND
   红表笔接 J2 Pin 1
   
2. R2在0% (最左):
   应读得 ≈ 0.8V
   
3. R2在50% (中间):
   应读得 ≈ 4.5V
   
4. R2在100% (最右):
   应读得 ≈ 8.5V
   
5. 缓慢旋转R2:
   电压应平滑变化,无跳跃
```

### 第2步: 频率测试

```bash
工具: 频率计或接收机

1. 连接频率计到FM天线
   或打开FM收音机调到空白台
   
2. 打开VCO模块电源
   
3. 缓慢旋转R2电位器
   
4. 观察频率变化:
   ├─ 应该有平滑的频率变化
   ├─ 变化范围应该是±2-5MHz
   └─ 没有明显的跳跃或抖动
   
5. 找到中心频率点:
   └─ 通常在R2中点附近
```

### 第3步: 波形质量测试

```bash
工具: 示波器 (可选)

1. 连接示波器到:
   ├─ CH1: J2 Pin 1 (控制电压)
   └─ GND: GND
   
2. 观察波形:
   ├─ 应该是干净的DC信号
   ├─ 不应有明显纹波
   ├─ 不应有噪声
   └─ 变化应该平滑
   
3. 如果有纹波:
   ├─ 增加C1容值(试试22uF)
   ├─ 或增加R3阻值(试试20K)
   └─ 或缩短连接线长度
```

### 第4步: 音质测试

```bash
工具: FM收音机

1. 调谐到FM台(88-108MHz)
   
2. 用VCO模块调节频率
   
3. 播放音乐,检查:
   ├─ 是否能清晰接收信号
   ├─ 是否有杂音
   ├─ 是否有失真
   └─ 全频率范围是否都能用
```

## 常见问题解决 (Troubleshooting)

### 问题1: 电压不变

```
原因排查:
□ R2电位器是否损坏?
  → 测量R2两端,应该0-100K可变
  
□ 连接是否正确?
  → 检查R2的3个脚都焊上了
  → 检查与R1的连接
  
□ 焊点是否冷焊?
  → 重新加热并补焊
```

### 问题2: 频率调不了

```
原因排查:
□ BB910是否焊接正确?
  → 用万用表测K脚和A脚的电阻
  → 应该能测到1-10欧(短接)
  
□ J3是否连接到正确位置?
  → 确认连接到FM发射机的C7点
  
□ 控制电压是否有变化?
  → 旋转R2,用万用表测J2 Pin 1
  → 应该能看到电压变化
```

### 问题3: 频率跳变

```
原因排查:
□ R2接触不好?
  → 清洁R2电位器
  → 转几圈涂上少量硅脂
  
□ 控制电压有纹波?
  → 增加C1滤波电容
  → 用示波器检查
  
□ 连接线干扰?
  → 缩短连接线
  → 使用屏蔽线
```

### 问题4: 杂音太大

```
原因排查:
□ 控制电压不够干净?
  → 增加C1(试试22uF)
  → 增加R3(试试20K)
  
□ 接地不良?
  → 检查所有GND连接
  → 用较粗的GND线
  
□ 电源纹波?
  → 在9V输入加一个10uF电解电容
```

## 校准和优化 (Calibration & Optimization)

### 频率线性度校准

```
1. 测量3个频率点:
   ├─ R2最左: 记录频率 F1
   ├─ R2中点: 记录频率 F2
   └─ R2最右: 记录频率 F3
   
2. 计算线性度:
   → (F2-F1) 应该 ≈ (F3-F2)
   
3. 如果不线性:
   ├─ 增大R1值 → 线性度变差
   ├─ 减小R1值 → 调范变小
   └─ 调整R3 → 精细调节
```

### 调范调节

```
想要更大的调范:
→ 减小R1值(试试47K)
→ 但线性度会变差

想要更小的调范:
→ 增大R1值(试试150K)
→ 会更线性,但调范减小
```

## 长期维护 (Maintenance)

```
□ 定期检查焊点,看是否松动
□ 如果长期不用,定期通电检查
□ 保存在干燥环境,避免潮湿
□ 避免阳光直射
□ 避免振动和碰撞
```

## 安全提示 (Safety)

```
⚠ 焊接时:
  □ 使用防静电地线
  □ 温度不超过350°C
  □ 时间不超过3秒/脚
  □ 避免过量焊锡
  
⚠ 使用时:
  □ 不要短路+9V和GND
  □ 不要用手触摸通电的焊点
  □ 不要在高温环境使用
  □ 不要长时间工作超过5小时
```
